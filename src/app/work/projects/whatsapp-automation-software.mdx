## Overview

Built an automation tool using PyWhatKit to schedule and send WhatsApp messages through APIs, paired with a modern React.js UI that increased engagement by ~30%.

## Engineering Details

- **API-Driven Automation**: Programmatic message scheduling and dispatch with robust error handling.
- **Secure by Design**: Enforced DevSecOps practices, Linux kernel security measures, and secure secrets handling.
- **Performance**: Leveraged multi-threading for concurrency and responsiveness.
- **CI/CD Ready**: Ensured compatibility with continuous integration and deployment workflows.

## Tech Stack

Python, PyWhatKit, FastAPI, React.js, Docker, GitHub Actions
