## Overview

Built a cross-platform voice assistant driven by AI, harnessing `pyttsx3` and NLP pipelines to automate tasks for desktop and mobile.

## Features

- **NLP + TTS**: Improved tokenization and response quality; used `pyttsx3` for high-quality text-to-speech.
- **IoT Integration**: Connected with IoT devices for ambient control and automation.
- **Cloud Native**: Deployed as a microservice via Docker on Azure using Azure Functions.
- **Quality**: Focused on testing and performance optimization for low-latency interactions.

## Tech Stack

Python, pyttsx3, spaCy/NLP, Docker, Azure Functions, React (companion UI)
